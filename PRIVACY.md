# Privacy Policy

This privacy policy applies to the **No Algorithms** browser extension.

## Data Collection
"No Algorithms" does **not** collect, store, transmit, or share any personal information, browsing history, or user data of any kind. 

## How it Works
All operations performed by this extension are executed completely locally on your device. The extension only injects CSS and JavaScript into specific supported websites (such as YouTube and Instagram) to hide algorithmic feeds. It does not communicate with any external servers.

## Third-Party Tracking
We do not use any third-party analytics, tracking tools, or external code. 

## Changes
If we make any changes to this privacy policy, we will update this document in the repository.

## Contact
If you have any questions or concerns regarding this privacy policy, please open an issue in the project repository.
