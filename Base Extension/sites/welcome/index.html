Welcome to No Algorithms

Your feeds are now hidden.

We have removed the explore and recommended feeds on YouTube and Instagram. No endless scrolling, just the content you actually search for.

We did not add an easy pause button, because we know how tempting it is.

Discipline is hard.

Browsers let you turn off extensions with a single click. When a craving hits, it is way too easy to just disable this block and start scrolling again.

If you really want to focus, you need to make it physically harder to disable this extension.

Locking it down (Browser)

Google Chrome makes it difficult to permanently lock a local extension. The easiest way to stop yourself is outside the browser:

  1. Install a system-wide blocker like Cold Turkey Blocker (Windows/Mac) or SelfControl (Mac).
  2. Block access to the chrome://extensions page entirely during your focus blocks so you can not click disable.
  3. Alternatively, use enterprise registry policies to lock it (advanced).

In Firefox, you can force the extension to stay on using enterprise policies:

  1. In your Firefox installation folder, create a new folder called distribution.
  2. Inside it, create a file named policies.json.
  3. Paste this code:
{
  "policies": {
    "ExtensionSettings": {
      "no-algorithms@example.com": {
        "installation_mode": "force_installed",
        "install_url": "file:///path/to/extension.xpi"
      }
    }
  }
}

Restart Firefox, and you will not be able to turn the extension off from the Add-ons menu anymore.
