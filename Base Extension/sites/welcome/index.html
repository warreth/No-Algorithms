Locking it down (Browser on OS)

In Firefox, you can force the extension to stay on using enterprise policies so the disable button becomes grayed out:

  1. Open your Firefox installation directory (typically C:\Program Files\Mozilla Firefox\).
  2. Create a new folder called distribution if it does not exist.
  3. Inside it, create a file named policies.json (you can use Notepad).
  1. Go to Applications, right-click Firefox, and select 'Show Package Contents'.
  2. Navigate to Contents/Resources/ and create a folder called distribution.
  3. Inside it, create a file named policies.json using TextEdit.
  1. Open your terminal and run: sudo mkdir -p /etc/firefox/policies (This creates the policy folder as an administrator).
  2. Create and open a file named policies.json inside that directory using a command-line editor like nano: sudo nano /etc/firefox/policies/policies.json. (Alternatively, log into your file manager as root to create the file).

To find your exact installation folder for any Firefox-based browser:

  1. Open a new tab and go to about:support
  2. Scroll down to the Application Basics section.
  3. Look for Application Binary to see exactly where your browser is installed.
  4. Create a distribution folder in that exact same directory (next to the browser executable).
  5. Inside it, create a file named policies.json.
  • Paste this exact JSON code inside policies.json:
{
  "policies": {
    "Extensions": {
      "Locked": ["no-algorithms@example.com"]
    }
  }
}

Restart Firefox, and you will not be able to turn the extension off from the Add-ons menu anymore.
