# No Algorithms by WarreTh

> This is a fork of [No-Algorithms](https://github.com/thejoeman23/No-Algorithms) by thejoeman23. 

A browser extension that removes short form content and algorithms from social media websites while keeping the main site intact.


## Features (YouTube & Instagram)
- Removed the home page and replaced with subscriptions page (only watch what you signed up for)
- Removed all Shorts or links to Shorts
- Removed all recommendations
- No off button or toggle 😲
- Fully modular logic for easily supporting new websites
- Supports Chrome **and** Firefox natively

## Installation
### Option 1: Firefox (Recommended)

You can easily install the extension directly from the official Firefox Add-ons store:
👉 **[Download on Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/no-algorithms/)**

> [!NOTE]
> If you want a pre-release version, download the `.xpi` file from the **[Releases](https://github.com/WarreTh/No-Algorithms/releases)** page and drag it directly into Firefox to install it.

### Option 2: Chrome (Recommended)

You can easily install the extension directly from the official Chrome Web Store:
👉 **[Download on Chrome Web Store](https://chromewebstore.google.com/detail/fcdiiaodicmddljfoceilmacapeppomi)**

> [!NOTE]
> If you want a pre-release version, you can install it manually by downloading the `chrome-extension-[VERSION].zip` file from the **[Releases](https://github.com/WarreTh/No-Algorithms/releases)** page. Unzip it, go to `chrome://extensions`, enable **Developer Mode**, and click **Load Unpacked** to select the folder.
